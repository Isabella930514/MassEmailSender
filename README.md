### Prerequisites

`python3`
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sinmentis/MassEmailSender
   ```
2. Configure Json files, follow the format with provided  files
3. Send Emails!
   ```sh
   cd src && python3 main.py
   ```
